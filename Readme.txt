Kidlogger v. 1.5.52 for Android 4 and newer.
Release date: 13 November 2015.

Records phone activity and all user actions on it (Android OS).

Main features of KidLogger 1.5.52:

1. The list of used apps and games with total time spent for each one;
2. The list of calls and SMS messages;
3. The list of created photos;
4. Phone coordinates for the day(requires Location function of the phone to be switched ON); 
5. The list of visited web sites(Internet browser);
6. Uploading the logs to your account on Kidlogger.net server.


***************************************************************
Installation 
***************************************************************

1. Install APK file ( KidLogger.apk ) (Settings, Security, Unknown sourses -> on)

2. Open KidLogger application

3. On the KidLogger Parental control control panel 
  - open settings > 
     - Switch all the features on;
     - Connect to kidlogger.net
         - enter your kidlogger.net account email; 
         Ensure connect status is "OK";
  - on the main screen click "Start Log"; 
  - click "Back" to leave KidLogger app running on the background.

Done!

***************************************************************
Removal
***************************************************************

1. Open Settings > Applications > find KidLogger in the list and click Uninstall
